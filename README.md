# Spell Scanner App

Prototype web mobile de lecture de sorts dessinés sur papier.

## Fichiers

- `index.html` : structure de l'application
- `style.css` : interface
- `script.js` : traitement d'image et reconnaissance simple des symboles

## Utilisation

1. Ouvrir `index.html` dans un navigateur.
2. Importer ou prendre une photo du sort.
3. Ajuster le seuil noir/blanc si besoin.
4. Cliquer sur `Analyser le sort`.

## Déploiement mobile simple

Mettre ces fichiers sur GitHub Pages, puis ouvrir le lien sur téléphone.
